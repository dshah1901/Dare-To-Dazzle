Dare to Dazzle is an application that showcases different types of jewellery. The user can choose from four provided categories of jewellery; “Rings, Earrings, Necklaces and Bracelets”. Each category redirects the user to a list of items of the specific type. DataProvider class to populate all the data, such as product name, description and cost. The user can choose one of the listed jewellery items and get more information regarding the particular item, including an image slider to switch between jewellery images. Moreover, the user can view the most clicked items on the landing page and search for any desired item to get more information about it.
Various ViewGroups have been utilized throughout the project for better visual presentations, such as RecyclerView, ListView and ScrollView.
For this project, Dhvani was Developer 1, and Pooja was Developer 2.

<hr>

## UI Design

The UI Design of this application is made keeping the Material Design Guidelines in mind.
It also is responsive for 5-7 inch devices in addition to landscape mode.
Animation like the Splash screen animation and transition effects between activities are used to make the app look more professional and enrich the UX.
